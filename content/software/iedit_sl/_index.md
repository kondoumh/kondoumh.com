今はなき Microsoft の RIA 技術 Silverlight で動作する iEdit クローンです。

- Windows 版では実現していないリンクへのリンクも作れます。
- フルスクリーンモードもあります。
- グラデーション付きで半透明のノードなどが作れます。
- アウトオブブラウザー機能でデスクトップアプリのように動作させることが可能です。
- GPU Acceleration により描画されます。
